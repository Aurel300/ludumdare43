Editor controls

  • Q – switch to game with current map
  • R – reset HP and MP of all units
  • Arrow key + A – add column / row with selected terrain
  • Arrow key + D – delete column / row
  • WSAD – move camera
  • T – import map from data
  • Z – export map to data
  • Space – switch editor mode:
    • Terrain mode:
      • IOP – select brush size
      • YXCVBN – select terrain type
    • Building mode, unit mode:
      • IOP – select player (0 = neutral)
      • YXCVB – select building or unit type

Game controls

  • O – switch to editor without saving map
  • P – switch to editor with current map
  • Space – End current player turn
